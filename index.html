Model Assumptions

Explicitly synthetic
  • Arrivals are deterministic from each truck post's configured loads-per-minute rate, using your estimated market-volume bands as the default calibration.
  • Scores are modeled as score percentiles on a stable 0-100 scale. A `75%` threshold means roughly the top `25%` of loads survive.
  • The proposed model keeps your local bucket threshold, then adds a global top-K cap so only the strongest immediately-workable loads remain actionable.
  • The dynamic model starts from the bucket threshold, then raises or lowers it using live loads per minute, expected passing flow for that lane, and total expected passing flow across all lanes.
  • A load counts as captured when an agent begins work and grabs the search before the stale window closes.
  • Agents remain occupied for the full handle time even after the grab is counted.
